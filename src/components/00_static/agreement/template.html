1. Object of Data Protection

Data protection serves to protect the privacy of individuals. It requires data processors to act lawfully, and confers enforceable rights to the affected persons with regard to the processing of their personal data. Section 3 of the Canton of Zurich’s Act on Information and Data Protection (Gesetz über die Information und den Datenschutz [IDG ZH]) defines personal data as any information related to an identified or identifiable individual. This includes details such as names, addresses, phone numbers, and e-mail or IP addresses.

2. Collecting, Processing, and Storing Data

In order to play the Lives in Transit game on this website, you will need to create a player profile. For this player profile we need to register the following:

  • e-mail address

In case of using the Universities Federated Login (AAI), the following extra attributes are requested:

  • Affiliation
  • Home organization

This data is processed in order to identify you in a class. At some point there may be content in the game which we are only allowed to distribute to members of specific organizations/affiliations, which is why these extra attributes are requested from the AAI system.

All other information that is given in is assumed to be fictional. The fictional information that is requested consist of:

  • First name
  • Last name

We process this information in order for the game to your `character` in the correct context.

Every time a Lives in Transit web page is requested or called up, the following access data are collected and stored in a web server file on UZH’s servers:

  • The IP address of the requesting computer (e.g. 123.456.97.36)
  • The site or address (URL) from which the web page was requested
  • The path and name of the requested web page
  • The date and time of the request (e.g. [12/Apr/2016:00:00:01 +0200])
  • The volume of data transferred
  • The access status
  • The type of access
  • A description of the type of web browser and/or operating system used
  • The session ID
  • The serial number of the requesting computer

These data are processed for the following purposes:

  • Safeguarding the network infrastructure and facilitating technical administration
  • Optimizing the use of specific web offerings
  • Identifying and tracking unauthorized access attempts

Web server log files are stored for six months after access is terminated. Once this period has elapsed, web server log files are automatically deleted, provided there has been no identified attack on the University’s network infrastructure necessitating the civil or criminal prosecution of the intruder and thus requiring that the log file be stored longer.

2a. Data Collection for Research and Teaching

While playing the Lives in Transit game, you will progress though a fictional world where you are asked to perform tasks. These tasks will have you enter information in the system. Your journey through the game-world and all related information you add to the system can be used for teaching and research purposes. A summary of information collected and processed this way:

  • The pages that you’ve visited while playing the game (your journey)
  • All information created in the game (notebook entries and exercises)

These data will be processed indefinitely, unless you request your account to be deleted. Accounts deleted in this way will not be restored.

3. Consent to Further Use of Data

Use of specific functions or services on the Lives in Transit website may require further collection, processing, and long-term storage of personal data such as names, addresses, and e-mail addresses. By entering and sending this data, you give your consent for this information to be processed.

4. Disclosure of Personal Data

Personal data will be disclosed to third parties (e.g. other authorities) only if required by binding legal provisions (e.g. the ruling of an authority, court orders) or for the purposes of legal or criminal prosecution (e.g. in the event of attacks on UZH’s network infrastructure). Personal data will not be disclosed to third parties for other purposes.

In the future, Lives in Transit may commission third-party service providers to process data collected via the Lives in Transit website for the purposes described above. Legal, technical, and organizational measures will be put in place requiring Lives in Transit and external service providers to comply with the relevant provisions of data protection law.

5. Tracking Settings

All your actions within the Lives in Transit game will be tracked. This is necessary for the game to function as intended. For teaching and research purpose it is necessary to know all the paths of all users. It is therefore possible to trace the results of this analysis back to a specific e-mail address and to monitor the behaviour of a data subject.

6. Cookies

So-called temporary cookies may be used to store session information of the game. Temporary cookies are small files that the Lives in Transit website stores on your computer to enable optimal use of the website; the cookies are deleted automatically when the web browser is closed. Please note that disabling, blocking, or deactivating cookies can limit the functionality of the Lives in Transit web pages.
